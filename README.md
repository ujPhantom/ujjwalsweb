# Personal website

A dependency-free personal portfolio starter. Open `index.html` directly in a browser.

## Personalize it

- Replace the sample name, bio, location, email, social links, and project details in `index.html`.
- Swap the three Unsplash image URLs for your own work or portrait images.
- Adjust colors and typography at the top of `styles.css`.

The page includes responsive navigation, scroll reveals, a live copyright year, and a click-to-copy email interaction.
